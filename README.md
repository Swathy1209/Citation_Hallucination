Citation Hallucination Detection in Large Language Models
Overview

Large Language Models (LLMs) sometimes generate fabricated or incorrect citations, a phenomenon known as citation hallucination. These hallucinated references may contain plausible author names, journal titles, or DOI patterns but do not correspond to real scholarly publications.

This project implements a citation hallucination detection framework that identifies suspicious references generated by LLMs using structural validation signals and external knowledge sources.

The system aggregates multiple hallucination markers into a heuristic suspicion score, enabling automated detection of fabricated citations in AI-generated text.

Methodology

The detection pipeline follows this process:

Citation Extraction → Validation → Similarity Analysis → Suspicion Scoring → Classification

1. Citation Parsing

Extract citation artifacts from text:

DOI identifiers

URLs

citation titles

Regular expressions are used to detect citation patterns.

2. DOI Validation (CrossRef API)

Generated DOIs are validated against the CrossRef registry.

Example:

import requests

def validate_doi(doi):
    url = f"https://api.crossref.org/works/{doi}"
    response = requests.get(url)
    return response.status_code == 200

If the DOI does not resolve in CrossRef, it is treated as a strong hallucination signal.

3. Metadata Similarity Check

The generated citation title is compared with official metadata retrieved from CrossRef.

from rapidfuzz import fuzz

def title_similarity(generated_title, crossref_title):
    score = fuzz.token_set_ratio(generated_title, crossref_title)
    return score / 100

Low similarity suggests that the citation may be fabricated or inconsistent.

4. URL Validation

Citation URLs are verified using HTTP requests to determine whether the referenced resource is accessible.

5. Near-Miss Citation Simulation

To simulate realistic hallucination patterns, near-miss citations are generated by modifying valid DOI identifiers.

Example:

Real DOI
10.1038/s41586-024-07421-0

Near-miss DOI
10.1038/s41586-024-07421-9

These cases mimic common LLM hallucination errors.

6. Suspicion Score

All hallucination markers are aggregated into a normalized suspicion score (0–1).

Higher scores indicate higher probability of hallucination.

Experimental Results

Final classification performance:

Metric	Value
Accuracy	92.88%
Precision	92.32%
Recall	96.13%
F1 Score	94.19%
ROC-AUC	~0.95

These results demonstrate strong capability in detecting hallucinated citations while maintaining low false positive rates.

Ablation Study

An ablation study was conducted to evaluate the contribution of different hallucination markers.

Feature Set	Accuracy	Precision	Recall	F1
DOI only	0.40	0.00	0.00	0.00
DOI + URL	0.60	0.60	1.00	0.75
DOI + Metadata	0.93	0.92	0.97	0.94
Full Model	0.93	0.92	0.96	0.94

The study shows that metadata similarity is the strongest hallucination detection signal, while DOI pattern checks alone are insufficient.

Error Analysis

Two primary error types were observed.

False Positives

Valid citations flagged as hallucinated due to:

incomplete metadata

unreachable URLs

shortened citation titles

False Negatives

Highly realistic fabricated citations containing:

plausible journal names

realistic author formatting

DOI-like identifiers
